# **Driver 28byj-48 (For Stepper Motor)**

| ----------------- | ---------------------- |
| +                 | 5V-2A (External PSU)\* |
| -                 | GND                    |
| IN1               | 5                      |
| IN2               | 19                     |
| IN3               | 18                     |
| IN4               | 21                     |
| IN5               | Not Connected          |
| IN6               | Not Connected          |
| IN7               | Not Connected          |
| Stepper Connector | Stepper Motor          |
