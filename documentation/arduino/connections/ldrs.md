# **LDR sensors (Light Intensity/Luminosity)**

- LDR 1 (Inside)

  | Sensor Pin              | Microcontroller Pin    |
  | ----------------------- | ---------------------- |
  | leg_1                   | 5V                     |
  | leg_2                   | GND (through resistor) |
  | Resistor-LDR connection | A0                     |

  | Sensor Pin              | Microcontroller Pin    |
  | ----------------------- | ---------------------- |
  | leg_1                   | 5V                     |
  | leg_2                   | GND (through resistor) |
