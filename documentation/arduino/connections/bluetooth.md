# **Bluetooth HC-05**

| Sensor Pin | Microcontroller Pin |
| ---------- | ------------------- |
| EN         | Not Connected       |
| VCC        | 3.3V                |
| GND        | GND                 |
| TXD        | D8                  |
| RXD\*      | D9                  |
| State      | Not Connected       |

\*RXD is also connected with D2 interrupt pin
